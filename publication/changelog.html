OpenPeppol eDEC Specifications Change Log

Update 2022-05-15

The file peppol-sml-types-v1.xsd was corrected.
The version of the code lists was updated from v8.0 to v8.1.

Update 2022-01-25

The old versions of the specification, that are no longer to be used, were removed. They can still be found in the GitHub repository.
The version of the code lists was updated from v7.5 to v8.0.
The migration guide of the previous changes were updated, to reflect the correct migration date (May 1st, 2022).

Release 2021-12-10

A general description on the changes is available in PDF format.
  • BusDoX Common Definitions → is no longer used
  • SMP update from v1.1.0 to v1.2.0
    • The changes have a 3 month period and enter into effect on March 10, 2022
    • A detailed migration guide for this change is available
  • SML update from v1.0.1 to v1.2.0
  • Peppol Policy for use of Identifiers update from v4.0.0 to v4.1.0
    • The changes have a 3 month period and enter into effect on March 10, 2022
    • A detailed migration guide for this change is available
  • Peppol Secure Transport Policy update from v1.0 to v1.1.0
  • Peppol AS4 Profile update from v2.0.0 to v2.0.2
  • Peppol SBDH Envelope Specification update from v1.2 to v1.2.1
  • Peppol Directory update from v1.1 to v1.1.1
Note: Previous specification changes were not documented like this.
